## Таблица "DataTable"

Таблица с данными.

**Параметры:**

- `header` (array) - Список колонок:
  - `title` (string) - Отображаемое название колонки
  - `alias` (string) - Алиас к которому строится соответсвие данных
- `body` (array) - Данные соответсвующие списку колонок:
  - `alias` (string) - Алиас к которому стоится соответсвие данных
  - `data` (any) - Содержание ячейки
- `showCheckboxColumn` (boolean) - Показывать колонку с чекбоксами

**Слоты:**

- `buttons` - Кнопки

**События:**

- `selectCharacter` - Отмечен чекбокс персонажа. Возвращает объект данных персонажа.
- `selectAllCharacters` - Отмечены все персонажи на странице. Возвращает массив данных персонажей.
- `addFavorite` - Персонаж добавлен в избранное. Возвращает массив данных пресонажей.
- `removeFavorite` - Персонаж удален из избранного. Возвращает массив id персонажей.
- `click` - Клик на имя персонажа. Возвращает объект персонажа.
